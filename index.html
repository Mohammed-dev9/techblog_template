Below code snippet is written in Java. 
                   
    package com.Code;
                                 
      public class Code{

      void methodCalling(){
        
        System.out.println("I have been called by the main() method.");
                         
    }
     
      public static void main(String[] args){

        new Code().methodCalling();
                         
    }
                        
    }

Explanation of code:

Every program in Java always starts with the main() method, so we should start analyzing the program from the main() method to understand what's actually happening.

We have a class called "Code", and it contains two methods:
1. The main() method, which is static.
2. A non-static or instance method called methodCalling().

We create an object of the "Code" class and call the non-static method methodCalling() using this object.

We know that an object has its own attributes and behaviors. Here, the object of the "Code" class has its behavior or property, i.e., the methodCalling() method, which is associated with the object.

Once you create an object and call the non-static method, all the statements inside that method will be executed. That is why we see the output: "I have been called."

Now you understand how to call a method. Go ahead and try it yourself!

Definitions of Different Technologies
Java

Java is an object-oriented programming language. It is used for building enterprise-level applications.

It can run on any operating system with its class file, which is why it is called Write Once, Run Anywhere (WORA).

Java is one of the most popular languages among developers around the world. Most legacy applications are built on Java.

It is secure, robust, and scalable. Everything revolves around objects, and objects have their own attributes and behaviors.
Java syntax is easy to read and understand.

Example: Syntax of a Variable

                                  

package com.Syntax;
                                 
  public class Syntax{
        

      int X=5;


    void syntaxPrinting(){

        
        System.out.println("Variable X value: "+X);
                         
    }
     
    public static void main(String[] args){

        new Code().syntaxPrinting();
                         
    }
                        
    }

The above is an example of how a variable is defined in Java.

HTML

HTML is a HyperText Markup Language. It is used for writing content for a web page, which is then displayed as plain text in browsers.

We can embed images, videos, links, and documents in our web page using HTML.

HTML has more than 5,000 elements for structuring web page content.

Example: Structure of an HTML document:

                    

    <!DOCTYPE HTML>
    <html lang="en-US">
    <head>
        <meta charset="UTF-8">
        <meta name="viewport" content="width=device-width, initial-scale=1.0">
    </head>
    <body>
        <header></header>
        <main>
            <section>This is a HTML document.</section>
        </main>
        <footer></footer>
    </body>
    </html>

Above is the basic structure of an HTML document.

CSS

CSS is a Cascading Style Sheet. It is used for designing a web page.

We can add colors, visuals, styles, and borders to our web page using CSS.

It basically makes a web page interactive, appealing, and user-friendly.

We can control the layout and design of the web page using CSS.

A CSS stylesheet is linked in the <head> section of the HTML document.

Example: CSS stylesheet

                    
    style{

          body{
                color:blue;
                background-color: black;
                font-size: 20px;
                font-weight:bold;
                }

        }

In the above example, this will add styles such as color, font size, and font weight to the <body> section in HTML.
