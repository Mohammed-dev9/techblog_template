Below code snippet is written in Java. 
                
                  package com.Code;
                                 
                    public class Code{

                         void methodCalling()
                         {

                                    System.out.println("I have been called.");
                         }

                         public static void main(String[] args)
                         {
                                           new Code().methodCalling();
                         }
                        
                      }

Explanation of code:

Every program in Java always start with main() method, so we should start analyzing the program from main() method to understand what's actually happening.
We have a class called "Code", and we have 2 methods inside it.
One is main() method, which is static and the other one is a non-static or an instance method.
We have created an object of "Code" class and we are trying to call the non static method (methodCalling()) with the help of its respective class object.
We know that an object has its own attributes and behavior. Here, the object of the class "code" has its behavior or property; i.e; methodCalling() method and it is associated with an object.
Once you create an object and call the non-static method, whatever statements present in that method will get executed. That is why, we're seeing the output as "I have been called."

Now you understood, how to call a method now go ahead and try yourself.

Definition on Different Technologies
Java

Java is an object oriented programming language. It is used for building enterprise-level applications.
It can be run on any operating system with it's class file i.e; it's called Write Once Run Anywhere (WORA).
Java is one of the most popular language among developers, around the world. Most of the legacy applications are build on Java.
It is secured, robust and scalable. Everything revolves around objects, and objects has it's own attributes and behavior.
Java syntax are easy to read and understand.

Syntax of a variable in Java:
int X = 5;

Above is an example of how a variable is defined in Java.

HTML

HTML is a hypertext markup language. It is used for writing the content for a web page, and it will display the plain text on browsers.
We can embed images, videos, links, & documents in our web page using HTML.
HTML has more than 5000 elements for structuring the web page content.

Example of a HTML document:

  
    <!DOCTYPE HTML>
    <html lang="en-US">
    <head>
        <meta charset="UTF-8">
        <meta name="viewport" content="width=device-width, initial-scale=1.0">
    </head>
    <body>
        <header></header>
        <main>
            <section>This is a HTML document.</section>
        </main>
        <footer></footer>
    </body>
    </html>

Above is a basic structure of a HTML document.

CSS

CSS is a cascading stylesheet. It is used for designing a webpage.
We can add colors, visuals, styles, & borders to our webpage using CSS.
It basically makes a webpage interactive, appealing & user-friendly.
We can control the layout & design of the webpage using CSS.
CSS stylesheet is link in head section of the HTML document.

Example of CSS:

                    
    style{

          body{
                color:blue;
                background-color: black;
                font-size: 20px;
                font-weight:bold;
                }

        }

In the above example, this will add some styles like color, font size & weight to the body section in HTML.
